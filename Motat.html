Motat Application

Unitec - 2nd Year - Semester 2 - 2013

This project was a team based project where we had an app supervisor, two programmers, one person in charge of the story and three 3-D modelers. I was one of the 3-D modelers and the program I used to create this plane was a program called SolidWorks. The other program I used were 3ds Max to do the texture map and Adobe Illustrator to draw the texture. This 3-D model was designed for a children’s game for Motat based in Auckland. The app/game is to teach kids the basics of how a plane can fly and to showcase Motat’s very own water plane (Short Solent Teal 1), this game shows the route where the plane traveled and allows the user to fly the plane. In other words it is a basic flight simulator game.
