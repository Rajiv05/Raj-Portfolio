Auckland City Yoobee Project
Overview

The aim of this project is to provide Auckland Council with a single page application that will allow visitors to the region to use a map-based interface to easily find destinations such as motels, restaurants and scenic spots. The user should be able to:


- View destinations
- Learn more about a destination
- Have the option to go to that destinationâ€™s website
- Search destinations by category
- Find the way to the destination from their current location on a mobile device

Colour Scheme & Typefaces

- Colours

As a team, we chose a very soft and minimal colour scheme with soft grey's, white, black, a soft blue and a soft purple which compliments the colours in the header images.

- Typefaces

Oswald is a great heading font because of how clean it looks in bold. Roboto is an elegant sans-serif typeface which works well with Oswald. Roboto's clean lines show that this font is very easy on the eye for a body of text.

Surveying

I conducted a survey through SurveyMonkey and sent it out to people where I got them to do certain things on the website. For example, I got them to go to the map section to filter through the different sections so that I can see how it is for users other than myself to use the website. By doing this I got valuable feedback and implemented the changes into my final design.

Quick Sketch Prototypes

A quick sketch of section layouts before prototyping using UXPin which is easy to prototype one page websites.

Main Features

Map

A map with a filter of options to view location websites and get directions using geo location, leaflet maps, google maps and API data.

Upcoming Events

A section of this app shows upcoming events so that it encourages people to go to them.

Popular

The idea of this app/website is that there should be a favourite's page where people can add locations to their favourites.

Responsive

Desktop, Ipad & mobile friendly.
